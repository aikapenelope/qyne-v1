# Market Data Sources Quick Reference

## Free / Freemium
| Source | Best For | URL |
|--------|----------|-----|
| Statista | Market size, consumer data | statista.com |
| Crunchbase | Startup funding, team size | crunchbase.com |
| World Bank | Macro economic data | data.worldbank.org |
| CEPAL | Latin America economic data | cepal.org |
| IDB | Latin America development data | iadb.org |
| Google Trends | Search interest over time | trends.google.com |
| SimilarWeb | Website traffic estimates | similarweb.com |
| BuiltWith | Technology adoption data | builtwith.com |

## Paid (search for cached/quoted data)
| Source | Best For |
|--------|----------|
| Gartner | Enterprise tech market analysis |
| McKinsey | Industry reports, digital transformation |
| a16z | Startup/VC market analysis |
| PitchBook | Detailed funding and valuation data |
| CB Insights | Startup market maps and trends |
| Grand View Research | Market size forecasts by industry |

## Site-Specific Search Patterns
```
# Find Statista data without paywall
"[topic] market size" site:statista.com

# Find McKinsey insights (often free summaries)
"[topic]" site:mckinsey.com/industries

# Find a16z analysis
"[topic]" site:a16z.com/content

# Find World Bank data for Latam
"[country] digital" site:data.worldbank.org
```
