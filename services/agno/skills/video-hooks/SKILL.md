# Video Hooks

The first 3 seconds determine if someone watches or scrolls. This skill
teaches you hook patterns that stop the scroll.

## Hook Categories (by retention rate)

### 1. Pattern Interrupt (highest retention)
Break expectations. Show something unexpected.
- Visual: zoom into something unusual, glitch effect, unexpected object
- Text: bold statement that contradicts common belief
- Example: "Los chatbots estan MUERTOS" (then explain why agents are different)

### 2. Curiosity Gap
Create a question the viewer NEEDS answered.
- "Esto es lo que pasa cuando automatizas WhatsApp por 30 dias"
- "Encontre algo en los datos de mi clinica que me preocupo"
- "La funcion de Aurora que nadie conoce pero todos necesitan"
- Rule: NEVER reveal the answer in the hook

### 3. Data Shock
Lead with a surprising number.
- "El 73% de los pacientes..." (specific, credible)
- "500 mensajes en 1 hora..." (scale that impresses)
- "De $0 a $3,000/mes en 90 dias..." (transformation)
- Rule: the number must be specific and verifiable

### 4. Direct Challenge
Challenge the viewer's current behavior.
- "Si todavia respondes WhatsApp manualmente, mira esto"
- "Tu clinica esta perdiendo pacientes y no lo sabes"
- "Dejen de usar Excel para manejar clientes"

### 5. Before/After
Show transformation visually.
- Split screen: manual process vs automated
- Time comparison: 20 minutes vs 2 seconds
- Works best with screen recordings or demos

## Hook Rules

1. **Text on screen in first frame** — many watch without sound
2. **Max 10 words** in the hook text
3. **No greetings** — never start with "Hola" or "Bienvenidos"
4. **No logos** in first 3 seconds — earn attention first
5. **Movement in first frame** — static images get scrolled past
6. **Spanish hooks outperform English** for Latam audience (obvious but often ignored)

## Hook + Content Pairing

| Hook Type | Best For Content About |
|-----------|----------------------|
| Pattern Interrupt | New product features, myth-busting |
| Curiosity Gap | Case studies, tutorials, behind-the-scenes |
| Data Shock | Industry trends, market reports, comparisons |
| Direct Challenge | Pain points, problem-solution content |
| Before/After | Product demos, transformation stories |

## Testing Hooks

For each content piece, generate 3 hook variants:
- Variant A: emotional/storytelling angle
- Variant B: data-driven/factual angle
- Variant C: bold/provocative angle

The creative director evaluates which has the strongest visual impact.
