# Whabi WhatsApp Message Templates

## Initial Contact
```
Hola {nombre}! Soy {agente} de {empresa}. 
Vi que estas interesado en {servicio}. 
Te puedo ayudar con mas informacion?
```

## Follow-up (No Response)
```
Hola {nombre}, te escribi hace unos dias sobre {servicio}. 
Quedo a tu disposicion si tienes alguna pregunta.
```

## Appointment Confirmation
```
Confirmado! Tu cita es el {fecha} a las {hora}. 
Te envio un recordatorio el dia anterior.
```

## Post-Service
```
Hola {nombre}! Como te fue con {servicio}? 
Tu opinion nos ayuda a mejorar. Gracias!
```

## Notes
- Always use formal "usted" for first contact
- Switch to informal "tu" only after client initiates
- Include business hours in signature
- Never send messages before 8am or after 8pm
